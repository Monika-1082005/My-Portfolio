My Work

Amazon Clone

Crafted an Amazon clone with HTML and CSS, emphasizing visual elements and UI. Enhanced front-end skills, structuring content and styling. Hands-on experience deepened understanding of responsive design, resulting in a user-friendly and aesthetically pleasing website.

Stopwatch Application

Precision in time management! My stopwatch, built with HTML, CSS, and JavaScript, offers seamless control - start, pause, and reset with ease. The sleek design enhances appeal, while lap time recording adds versatility. Ideal for workouts or races, this responsive application is your go-to for accurate timekeeping.

Weather Application

Explore real-time weather with my Weather application, crafted using HTML, CSS, and JavaScript. Fetching data from a reliable API based on your location or input, it provides current conditions and temperature. The sleek design ensures an easy, informative experience, making it your go-to for up-to-date weather insights. Stay ahead with precision meteorological information at your fingertips.

Starbucks Landing Page

Check out my Starbucks-inspired landing page! Crafted with HTML and CSS, it's responsive and captures the brand essence. From familiar colors to layout, it mirrors the Starbucks aesthetic, showcasing my web development skills and commitment to a seamless user experience. Welcome to a caffeine-infused digital journey!

Tic Tac Toe Game

I crafted a Tic Tac Toe game using HTML, CSS, and JavaScript. The game boasts a sleek design with a 3x3 grid layout, created using CSS grid. JavaScript handles the game's mechanics, enabling smooth player turns, win checks, and symbol updates. By listening to player clicks, the interactive interface ensures an engaging experience. Responsive design ensures seamless gameplay across different devices.
