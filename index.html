My Work

Event Calendar

"Dynamic Event Calendar" is a simple, efficient tool for managing monthly events. It uses local storage to save events across sessions and lets users search, organize, and export events in JSON format for easy sharing.

AwareHer

"AwareHer" is a platform dedicated to empowering women by providing financial education tools to build literacy and confidence, guidance on legal rights to help women advocate for themselves, easy access to policies relevant to women, and resources for self-defense training to enhance safety.

SweatFlix

"SweatFlix" is your home workout assistant, featuring a chatbot for solving doubts, workout videos, the latest news and trends, and gamified challenges. This hackathon project is built with HTML, CSS, and JavaScript for the frontend, and Express.js and MongoDB for the backend.

Amazon Clone

I crafted an Amazon clone using HTML and CSS, focusing on visual elements and UI. This project enhanced my front-end skills by improving content structuring and styling. Through hands-on experience, I deepened my understanding of responsive design, resulting in a user-friendly and aesthetically pleasing website.

Stopwatch Application

Precision in time management! My stopwatch, built with HTML, CSS, and JavaScript, offers seamless control with start, pause, and reset functionality. The sleek design enhances appeal, while lap time recording adds versatility. Ideal for workouts or races, this responsive application ensures accurate timekeeping.

Weather Application

Explore real-time weather with my Weather Application, crafted using HTML, CSS, and JavaScript. Fetching data from a reliable API based on your location or input, it provides current conditions and temperature. The sleek design ensures an easy, informative experience, making it your go-to for up-to-date weather insights.

Starbucks Landing Page

Check out my Starbucks-inspired landing page! Crafted with HTML and CSS, it’s fully responsive and captures the brand’s essence. From familiar colors to layout, it mirrors the Starbucks aesthetic, showcasing my web development skills and commitment to a seamless user experience.

Tic Tac Toe Game

I crafted a Tic Tac Toe game using HTML, CSS, and JavaScript. The game features a sleek 3x3 grid layout created with CSS Grid, while JavaScript handles game mechanics, ensuring smooth player turns, win checks, and symbol updates. By listening to player clicks, the interactive interface delivers an engaging experience.
