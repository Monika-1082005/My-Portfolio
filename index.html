My Work

Amazon Clone

I developed an Amazon clone using only HTML and CSS, focusing on recreating the platform's visual elements and user interface. This project allowed me to enhance my front-end skills, including structuring content with HTML and styling it using CSS. Through this hands-on experience, I gained a deeper understanding of responsive design and layout principles, enabling me to create a user-friendly and aesthetically pleasing website.

Wanderers Point

Welcome to Wanderers Point, your ultimate travel destination guide! Crafted with love using HTML and CSS, our website offers a seamless and visually captivating experience for all travel enthusiasts. Hosted on the reliable Microsoft Azure platform, Wanderers Point ensures lightning-fast performance and secure access, inspiring your wanderlust and making your travel dreams a reality, one click at a time. Start exploring today and let your imagination take flight with Wanderers Point!

Tic Tac Toe Game

I crafted a Tic Tac Toe game using HTML, CSS, and JavaScript. The game boasts a sleek design with a 3x3 grid layout, created using CSS grid. JavaScript handles the game's mechanics, enabling smooth player turns, win checks, and symbol updates. By listening to player clicks, the interactive interface ensures an engaging experience. Responsive design ensures seamless gameplay across different devices.
